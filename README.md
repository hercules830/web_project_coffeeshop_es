# Triple Espresso

1. Triple Espresso - Página Web de Cafetería

Este proyecto consiste en el desarrollo de una página de aterrizaje (landing page) para una cafetería de especialidad llamada Triple Peaks. El sitio ha sido construido siguiendo un diseño profesional (brief) enfocado en la claridad, la estética y la experiencia de usuario.

2. Descripción y Funcionalidad
La página tiene como objetivo presentar la cafetería y facilitar la interacción con los clientes. Sus funciones principales son:
Navegación Intuitiva: Menú con anclajes internos que dirigen a las secciones de reservación y contactos.
Presentación Visual: Un encabezado atractivo que utiliza imágenes de fondo y elementos decorativos posicionados estratégicamente.
Sistema de Reservas: Un formulario funcional con validaciones nativas, campos de entrada personalizados y una casilla de verificación de términos de uso diseñada a medida.
Pie de Página (Footer): Espacio informativo con enlaces a redes sociales e iconos alineados, además de información legal y de derechos de autor.

3. Tecnologías y Técnicas Utilizadas
Para el desarrollo de este sitio se emplearon las siguientes herramientas y métodos:
HTML5 Semántico: Uso de etiquetas descriptivas como header>, main>, section>, fieldset>, nav> y footer> para mejorar la accesibilidad y el SEO.
CSS3 Moderno:
Metodología BEM: Organización de clases para un código escalable y fácil de leer.
Flexbox: Implementado para la alineación de menús, formularios y elementos del footer.
Posicionamiento (Absolute/Relative): Utilizado para capas de diseño complejas, como la imagen principal del encabezado y el círculo decorativo del footer.
Box Model (Border-box): Aplicado para garantizar que los campos de entrada mantengan un tamaño exacto de 60px de altura incluyendo rellenos y bordes.
Personalización de Formularios: Uso de appearance: none para rediseñar casillas de verificación (checkboxes) desde cero.
Transiciones (Transitions): Efectos suaves en los estados hover de botones y enlaces para mejorar la interactividad.
Fuentes de Google Fonts: Integración de las familias tipográficas Inter y Noto Serif para una jerarquía visual clara.
Normalización: Uso de normalize.css para asegurar la consistencia del diseño en diferentes navegadores.
